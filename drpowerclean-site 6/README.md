# D&R Powerwashing Website

Local student powerwashing business — New Hope, PA.

## Project Structure

```
drpowerwashing-site/
├── index.html       # Main page
├── style.css        # All styles
├── script.js        # Sliders + email logic
└── images/
    ├── logo.png     # D&R logo (transparent bg)
    ├── hero1.jpg    # Hero slider image 1  ← ADD YOUR OWN
    ├── hero2.jpg    # Hero slider image 2  ← ADD YOUR OWN
    ├── hero3.jpg    # Hero slider image 3  ← ADD YOUR OWN
    ├── before1.jpg  # Before/after card 1 - before
    ├── after1.jpg   # Before/after card 1 - after
    ├── before2.jpg  # Before/after card 2 - before
    └── after2.jpg   # Before/after card 2 - after
```

## Setup

1. Add 3 hero images to `images/` named `hero1.jpg`, `hero2.jpg`, `hero3.jpg`
   - Use powerwashing action shots or clean driveway/patio photos
   - Recommended size: 1400×800px

2. Push to GitHub and connect to Vercel for auto-deploy
